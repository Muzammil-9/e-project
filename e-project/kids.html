KICK THE CAN RULES

1. Kick the Can is a game that has been popular since the 1930s.
2. It combines tag, hide and seek, and capture the flag into one exciting game
3. Any age group can enjoy this game, and it can be played with as few as 3 or over 20 players.
4. [1] To get your game of Kick the Can started off right, you'll need to organize players and the field, doing things like establishing boundaries and setting up a jail for caught players.
5. After that, playing the game is relatively simple, and there are variations you can use to keep this game fresh.
